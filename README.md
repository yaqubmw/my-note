# Welcome to My Note Apps
**My Note Apps** ini dirancang dengan Java Spring Boot dan terhubung dengan basis data PostgreSQL. **My Note Apps** dapat mengoperasikan CRUD.

## My Note App Endpoints

| Method | Endpoint |
|--|--|
| POST | `/api/v1/notes` |
| GET | `/api/v1/notes` |
| GET | `/api/v1/notes/{id}` |
| PUT | `/api/v1/notes/{id}` |
| DELETE | `/api/v1/notes/{id}` |
